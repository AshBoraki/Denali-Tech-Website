DTNT FAQ

Answers for the install, update, and Pro questions customers ask first.

This FAQ keeps the commercial path simple. It covers the practical questions around Windows support, Store delivery, Pro activation, and what to expect when using DTNT in the field.

Does DTNT work on Windows only?

Yes. DTNT is a Windows desktop app built for Windows 10 and Windows 11.

Should I install DTNT from Microsoft Store or the website?

Microsoft Store is the preferred install path. The website path remains available as a direct installer and backup path.

Can I use DTNT before buying Pro?

Yes. DTNT keeps the core workflows available so you can use the app first and move to Pro when you need saved detail, exports, or activation.

How does Pro activation work?

Buy Pro through the DTNT checkout, then use the purchase email and activation code from the delivery email inside the Profiles workspace.

Does DTNT have a portable backup path?

Yes. The product page keeps a portable fallback available when the Store route is not the right fit for the job or machine.

What does Pro unlock?

Pro unlocks saved Wi-Fi detail reveal, exports, saved-result workflows, and the commercial activation path built for cleaner handoff and repeat use.
