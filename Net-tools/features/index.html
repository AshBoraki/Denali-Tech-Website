DTNT feature guide

One Windows toolset for the network work you actually do.

DTNT brings Wi-Fi visibility, IP discovery, speed testing, saved profile review, and operator-facing diagnostics into one desktop app so you can inspect the network without bouncing between five utilities.

Core workflows

  • Survey nearby Wi-Fi radios and compare signal posture without guesswork.
  • See channel pressure and lane overlap before you recommend changes.
  • Discover IP devices, filter web targets, and review network identity context.
  • Run and keep speed-test history for repeat checks and handoff evidence.
  • Inspect saved Wi-Fi profiles and compare live-versus-saved security state.

Wi-Fi analyzer

Review nearby radios, see signal posture, group the strongest access points by band, and keep the live survey surface readable enough for real field use.

Channel analysis

Focus on one band at a time and compare occupied lanes without the usual clutter of overlapping charts that hide the actual decision.

IP scanner

Discover network devices, filter inventory by infrastructure or endpoints, and surface browser-access targets fast when you need to sign in to local equipment.

Speed test

Track latency, download, upload, and saved history so you can compare runs, keep evidence, and export what matters later.

Profiles

Review Windows saved Wi-Fi profiles, compare live security posture, and keep activation and release details in the same place.

Operator polish

The DTNT shell is tuned to read like desktop software instead of a toy dashboard, with dark and light themes, tighter control states, and denser technical surfaces.
