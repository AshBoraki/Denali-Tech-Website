# Denali Tech Website

Static local copy of the live Denali Tech site at https://denalitechs.com/.

## Local Preview

From this folder:

```sh
python3 -m http.server 8080 --bind 127.0.0.1
```

Then open:

```text
http://127.0.0.1:8080/
```

## Notes

- This project was mirrored from the live site on 2026-06-01.
- The current baseline is static HTML, CSS, JavaScript, images, and video assets.
- Shared mobile menu files are available at clean local paths:
  - `assets/js/mobile-menu.js`
  - `assets/css/mobile-menu.css`
