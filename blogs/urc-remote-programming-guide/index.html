You might be surprised by how much your URC remote can do when it's properly programmed. Here's what we can set it up to control:

  • Smart Home Devices: Smart lights (Lutron, Philips Hue), smart thermostats (Nest, Ecobee), motorized shades, smart locks
  • Climate Control: HVAC systems, ceiling fans, fireplaces
  • Security Systems: Alarm systems, security cameras, door locks
